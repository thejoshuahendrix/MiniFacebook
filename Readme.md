# Mini Facebook 
## by Joshua Hendrix

This is a facebook like clone with picture and video uploading.

It has user authentication using MongoDB

To run: 

`git clone` the repo

Replace env-example.txt with your .env and credentials for MongoDB Atlas and your JWT secret

`cd server`

`yarn install`

`cd ../ && cd client`

`yarn install`

Open two terminals

In 1st

`cd server`

`yarn start`

In 2nd

`cd client`

`yarn start`
